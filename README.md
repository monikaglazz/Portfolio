# Portfolio
Welcome to My Portfolio! This repository serves as a hub for a diverse range of projects and analyses, encompassing SQL queries, database projects, and data analyses utilizing various Python libraries.

## Getting Started
Explore the various projects and analyses in the repository to gain insights into SQL, database design, and data analysis with Python.

### How to Navigate:
- Browse the individual folders to access specific projects, SQL analyses, and Python-based data analyses.
- Each project or analysis is accompanied by code snippets for easy comprehension.
 
## Repository Structure

### SQL Analyses

Explore a collection of SQL analyses that delve into different datasets, providing valuable insights and uncovering patterns within the data. These analyses showcase the power of SQL for data extraction and manipulation. Each analysis is accompanied by relevant comments and explanations.

### Database Projects

Dive into meticulously designed database projects, each tailored to address specific use cases and scenarios. These projects demonstrate proficiency in database architecture, normalization, and optimization, ensuring efficient data management and retrieval.

1. [**Hospital Database System**](https://github.com/monikaglazz/Portfolio/tree/main/Hospital%20Database%20System)
   - Description: A comprehensive database system tailored for hospitals, streamlining patient records, appointments, and medical staff management for efficient healthcare operations.
   - Files: `hospital_db.sql`, `hospital_db_diagram.dbs`, `hospital_db_diagram.png`, `roles_privileges.sql`
  
2. [**Blood Donations Database**](https://github.com/monikaglazz/Portfolio/tree/main/Blood%20Donations%20Database)
   - Description: A robust database for blood donation centers, enabling seamless tracking of donor information, blood inventory, and facilitating life-saving transfusions with accuracy and speed.
   - Files: `bdd_schema.dbs`, `bdd_schema.png`, `data_for_db.sql`, `database_file.sql`, `schema.diagram`

3. [**College Database**](https://github.com/monikaglazz/Portfolio/tree/main/College%20Database)
   - Description: A dynamic database system for educational institutions, managing student enrollment, course registration, grades, and faculty records to enhance administrative efficiency.
   - Files: `col_db_data.sql`, `college_db.sql`, `college.diagram`, `college_schema.png`

4. [**Online Shop Database**](https://github.com/monikaglazz/Portfolio/tree/main/Online%20Shop%20Database)
   - Description: An e-commerce database, handling product listings, customer orders, payment processing, and inventory management, ensuring a seamless online shopping experience for users.
   - Files: `shop_db.sql`, `shop_db_data.sql`, `schema.diagram`, `osd_schema.png`
  
### Data Analyses with Python

Discover a series of data analyses leveraging popular Python libraries. These analyses encompass a wide array of topics and datasets, showcasing the versatility of Python in exploratory data analysis and visualization.

1. [**Risk and Returns: The Sharpe Ratio**](https://github.com/monikaglazz/Portfolio/tree/main/Risk%20and%20Returns%3B%20The%20Sharpe%20Ratio)
   - Description: This project applies the Sharpe Ratio to real financial data using pandas, helping assess risk-adjusted returns for investment decisions.
   - Files: `analysis.ipynb`, `stock_data.csv`, `benchmark_data.csv`, `iris.csv`

2. [**Analyzing TV data**](https://github.com/monikaglazz/Portfolio/tree/main/Analyzing%20TV%20data)
   - Description: Explore Super Bowl Data to gain insights into game outcomes, viewership and halftime shows.
   - Files: `halftime_musicians.csv`, `super_bowls.csv`, `tv.csv`, `analysis.ipynb`
  
3. [**Investigating Netflix movies and guest stars in The Office**](https://github.com/monikaglazz/Portfolio/tree/main/Investigating%20Netflix%20movies%20and%20guest%20stars%20in%20The%20Office)
   - Description: Uncover trends in Netflix movie lengths and identify top guest stars in the most popular episode of "The Office" using a combination of lists, loops, pandas, and matplotlib.
   - Files: `netflix_data.csv`, `analysis.ipynb`, `color_data.csv`
